# 🌐 Landing Page - Desenvolvimento Web I

## 📖 Sobre o projeto

Este projeto foi desenvolvido como atividade avaliativa da disciplina de Desenvolvimento Web I, com o objetivo de construir uma página estática utilizando HTML5 e CSS3, aplicando boas práticas de estruturação semântica, organização de código e Flexbox para composição do layout.

A proposta consistiu na criação de uma landing page moderna, priorizando organização visual, reutilização de estilos e padronização dos componentes, sem utilização de JavaScript.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox

---

## 📚 Conceitos aplicados

- Estrutura semântica em HTML5
- Organização de arquivos
- Flexbox para alinhamento e posicionamento
- Boas práticas de CSS
- Reutilização de classes
- Variáveis CSS (`:root`)
- Hierarquia de títulos
- Acessibilidade básica
- Organização de layout
