# 🍲 Sistema de Gestão de Soparia

## 📖 Sobre o projeto

Este projeto foi desenvolvido como atividade avaliativa da disciplina de Desenvolvimento Web I, com o objetivo de criar um sistema web para gerenciamento de uma soparia utilizando apenas HTML5 e CSS3.

O sistema simula um ambiente real de gerenciamento de pedidos, contendo autenticação visual, gerenciamento de mesas, controle de pedidos e interface responsiva, aplicando conceitos modernos de desenvolvimento Front-end sem utilização de frameworks.

## 🎯 Objetivos da atividade

- Desenvolver um sistema web composto por múltiplas páginas.
- Aplicar HTML5 semântico.
- Construir layouts utilizando Flexbox e CSS Grid.
