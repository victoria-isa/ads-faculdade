# 🗄️ Exercícios de SQL

## 📖 Sobre o projeto

Este repositório reúne os exercícios desenvolvidos durante a disciplina de Banco de Dados do curso de Análise e Desenvolvimento de Sistemas (IFPE).

O objetivo é praticar consultas SQL, reforçando conceitos fundamentais para manipulação, filtragem e análise de dados, utilizando a linguagem SQL em diferentes cenários.

---

## 🚀 Tecnologias utilizadas

- SQL
- Banco de Dados Relacional

---

## 🎯 Objetivos

- Praticar comandos SQL.
- Desenvolver raciocínio lógico para consultas em bancos de dados.
- Aprender técnicas de filtragem e organização de informações.
- Consolidar conhecimentos em bancos de dados relacionais.

---

## 💡 Aprendizados

Durante os exercícios foi possível desenvolver habilidades como:

- Construção de consultas SQL;
- Manipulação de dados utilizando filtros;
- Ordenação e agrupamento de informações;
- Utilização de funções de agregação;
- Escrita de consultas mais legíveis e organizadas;
- Desenvolvimento do raciocínio lógico aplicado à análise de dados.
