# Como contribuir com este repositório

Este repositório é o **modelo** usado para gerar os repositórios individuais dos alunos. Se você encontrar um erro no enunciado, algum arquivo `.sql` com problema ou qualquer inconsistência, **abra uma issue** neste repositório (o template, não o seu fork pessoal).

## Antes de abrir uma issue

- Verifique se o problema já não foi relatado por outro colega (busque nas issues existentes).
- Certifique‑se de que o erro está no template original e não apenas no seu repositório pessoal.
- Abra a issue no repositório de template da disciplina (pbd-atividades) não no fork que você recebeu.

## Como abrir uma boa issue

Use um título claro e descreva o problema com detalhes.

### Exemplo

```
Título: Erro no exercício 3 – condição WHERE incorreta

No arquivo `enunciado-lista-NN.md`, o exercício 3 pede "clientes que compraram em 2023", mas a tabela de pedidos só contém dados até 2022. Sugiro alterar o ano para 2022.
```

## Importante

Issues são **apenas para reportar problemas no material** (enunciados, scripts, links). Dúvidas sobre como resolver os exercícios ou sobre SQL devem ser levadas aos canais de comunicação da disciplina (fórum, e‑mail, etc.).

Obrigado por sua colaboração!
