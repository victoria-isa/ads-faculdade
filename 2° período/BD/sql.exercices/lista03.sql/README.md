# Lista de Exercícios – Banco de Dados

Este repositório contém o enunciado e os arquivos‑resposta para uma lista de exercícios da disciplina **Banco de Dados** (2º período).

## Estrutura

- `enunciado-lista-XX.md` – descrição dos exercícios.
- `resp-lstXX-exNN.sql` – arquivos onde você deve escrever suas soluções (um por exercício).
- `.gitignore` – ignora arquivos desnecessários.

```
bd-lista-03/
├── README.md                    ← readme do repositório template
├── README_ESTUDANTE.md          ← readme do fork do estudante
├── CONTRIBUTING.md              ← guia de contribuição
├── enunciado-lista-03.md        ← enunciado com todos os exercícios
├── .gitignore                   ← lista de arquivos e diretórios que não devem ir para o github.
├── sql/
│   ├── loja_schema.sql          ← cria as tabelas do banco de dados
│   └── loja_seed.sql            ← popula o banco com dados de exemplo
└── respostas/                   ← lista de exercícios 03
│   ├── resp-lst03-ex01.sql      ← sua resposta para o exercício 01
│   ├── resp-lst03-ex02.sql      ← sua resposta para o exercício 02
│   └── resp-lst03-exNN.sql      ← (cada lista tem uma quantidade variada de exercícios)
```


## Uso

Este é o **repositório template**. Cada aluno receberá uma cópia individual através do processo automático da disciplina. Instruções detalhadas para clonar, resolver e entregar estão no arquivo `README_ESTUDANTE.md` (dentro do seu repositório pessoal).

## Feedback

Se você encontrar algum erro no enunciado ou nos arquivos de apoio, abra uma **issue** neste repositório seguindo as orientações do [`CONTRIBUTING.md`](CONTRIBUTING.md).
