# ads-faculdade
# Análise e Desenvolvimento de Sistemas

Este repositório reúne os projetos, atividades, exercícios e materiais desenvolvidos ao longo da graduação em Análise e Desenvolvimento de Sistemas.

O conteúdo está organizado por período letivo e por disciplina, permitindo acompanhar a evolução dos estudos e o desenvolvimento das habilidades adquiridas durante o curso.

## Estrutura

```text
ads-faculdade/
│
├── 1-periodo/
├── 2-periodo/
├── 3-periodo/
├── 4-periodo/
├── 5-periodo/
└── ...
```

Cada período contém as disciplinas cursadas, com seus respectivos projetos, exercícios, anotações e materiais de apoio.

## Objetivo

Este repositório tem como finalidade documentar minha trajetória acadêmica, organizar os trabalhos realizados durante a graduação e servir como um registro da evolução dos conhecimentos adquiridos.

## Conteúdo

* Projetos desenvolvidos em sala de aula
* Exercícios propostos pelos professores
* Trabalhos acadêmicos
* Estudos complementares
* Materiais de apoio
* Documentação dos projetos

## Tecnologias

As tecnologias presentes neste repositório variam conforme as disciplinas cursadas e poderão incluir linguagens de programação, bancos de dados, ferramentas de desenvolvimento e outras tecnologias utilizadas ao longo da graduação.
